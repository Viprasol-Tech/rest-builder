# Rest Builder

Build REST APIs quickly

## Features

- Zero external dependencies (stdlib only)
- Easy-to-use CLI interface
- Professional Python implementation
- MIT licensed

## Installation

```bash
pip install -e .
```

Or clone and install:

```bash
git clone https://github.com/Viprasol-Tech/rest-builder
cd rest-builder
pip install -e .
```

## Usage

### Python

```python
from rest_builder import RestBuilder

result = RestBuilder.process("data")
print(result)
```

### CLI

```bash
python -m rest_builder "your input here"
```
